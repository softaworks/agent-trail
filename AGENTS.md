# AgentTrail

Local web viewer for Claude Code session history across multiple directories.

## Quick Reference

- **Runtime:** Bun
- **Install:** `bun install`
- **Dev:** `bun run dev`
- **Start:** `bun run start`
- **Build:** `bun run build`
- **Typecheck:** `bun run typecheck`
- **Lint:** `bun run check`
- **Format:** `bun run format`
